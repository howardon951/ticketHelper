# KKTIX 搶票助手

自動化搶票工具，幫你完成選票、勾選同意、按下一步的重複操作。

> **重要提示**  
> 本工具僅供輔助，不保證一定能買到票。

---

## 安裝步驟

### 1. 安裝 Tampermonkey

根據你的瀏覽器安裝 Tampermonkey：
- [Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepogocplenbaon)
- [Firefox](https://addons.mozilla.org/firefox/addon/tampermonkey/)

### 2. 匯入腳本

1. 點擊 Tampermonkey 圖示 → 「建立新腳本」
2. 刪除預設代碼
3. 複製 `ticket_assistant.user.js` 的全部內容貼上
4. 按 `Ctrl+S` 儲存

## 使用方式

### 方式一：完整自動化

1. 前往任意 KKTIX 頁面（腳本會自動載入）
2. 在右上角的控制面板：
   - 輸入活動網址（例如 `https://ntw.kktix.cc/events/ntw-copy-2`）
   - 選擇票種（第幾個）
   - 選擇張數
3. 點擊「**開始搶票**」
4. 腳本會自動：前往購票頁 → 選票 → 勾選 → 下一步

### 方式二：測試模式

點擊「**測試（不按下一步）**」可以：
- 測試選票和勾選是否正常
- 不會自動點擊下一步
- 適合先確認設定是否正確

## 功能說明

| 功能 | 說明 |
|------|------|
| 輸入活動 URL | 支援 `xxx.kktix.cc` 和 `kktix.com` 格式 |
| 票種選擇 | 選擇第 1~4 個票種 |
| 張數選擇 | 1~4 張 |
| 開始搶票 | 自動完成所有步驟並點擊下一步 |
| 測試模式 | 只執行選票和勾選，不自動下一步 |

## 常見問題

### Q: 腳本沒有出現？
A: 確認 Tampermonkey 已啟用，並檢查腳本的 `@match` 是否包含當前網址。

### Q: 票種選錯了？
A: 使用「測試模式」先確認票種順序是否正確。

### Q: 下一步按鈕沒有自動點擊？
A: 可能是網路延遲，腳本會等待按鈕啟用後再點擊。
