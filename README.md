# 🎫 拓元 tixCraft 搶票助手 使用手冊

這是一份專為新手設計的教學，教你如何安裝並使用這個「半自動」搶票輔助腳本。

> [!IMPORTANT]
> **本工具僅供輔助，不保證一定能買到票。**
> 它的作用是幫你完成「選區域、填張數、勾同意」等重複動作，最關鍵的「驗證碼」仍需由你親手輸入。

---

## 🛠 第一步：安裝環境 (只需做一次)

1.  **安裝瀏覽器擴充功能**：
    - 請根據你的瀏覽器安裝 **Tampermonkey (油猴)**：
      - [Chrome 線上應用程式商店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
      - [Edge 擴充功能](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepogocplenbaon)
2.  **確認安裝成功**：安裝後，瀏覽器右上角會出現一個黑色像正方形的圖示 🧩。

---

## 📝 第二步：匯入腳本

1.  點擊瀏覽器右上角的 Tampermonkey 圖示，選擇 **「建立新腳本」 (Create a new script)**。
2.  將原本編輯器裡預設的所有代碼全部刪除。
3.  打開本專案中的 `ticket_assistant.user.js` 檔案，複製裡面**所有的代碼**。
4.  貼回到 Tampermonkey 的編輯視窗中。
5.  點擊左上角的 **「檔案」 (File) > 「儲存」 (Save)**。

---

## 🚀 第三步：開始搶票

1.  **進入拓元官網**：打開你想搶的演唱會售票頁面。
2.  **觀察介面**：頁面右手邊應該會出現一個深色的 **「TIXCRAFT ASSISTANT」** 小視窗，顯示 `Scanning...`。
3.  **自動化流程**：
    - **選區頁面**：腳本會自動幫你點擊第一個「還有票」的區域。
    - **填單頁面**：
      - 自動選好 **2 張票**（可在腳本中修改數量）。
      - 自動勾選底下的 **「我同意服務條款」**。
      - **自動聚焦驗證碼**：你的游標會自動停在驗證碼輸入框，你只需看圖並輸入英文字母。
4.  **手動按下送出**：輸入完驗證碼後，請手動按下 Enter 或點擊「立刻購票」。

---

## ⚙️ 進階修改 (如何改張數？)

如果你想買 1 張或 4 張票，請依照以下步驟修改：

1.  打開 Tampermonkey 控制面板，點擊腳本旁邊的「編輯」。
2.  尋找第 **48 行** 左右的這段代碼：
    ```javascript
    select.value = "2"; // 這裡的 2 就是張數
    ```
3.  把 `"2"` 改成 `"1"` 或 `"4"` (注意要保留引號)。
4.  按下 `Ctrl+S` (或 `File > Save`) 儲存。

---

## ⚠️ 安全注意事項

- **STOP 按鈕**：如果你發現網頁怪怪的，點擊輔助小視窗上的 **STOP** 紅按鈕，腳本就會立刻停止所有動作。
- **不建議全自動**：本腳本刻意不自動提交表單，是為了降低被系統偵測為機器人的風險，保護你的帳號不被封存。

祝你搶票成功！🍀
